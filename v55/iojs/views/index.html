XIVSync API

Server Location: Tokyo

Character

/character/search

  • Parameter:   name - /character/search?name=premium
  • Parameter:   server - /character/search?server=adamantoise

/character/get/{id}

  • Example:   /character/get/730968
  • Parameter:   restrict - /character/get/730968?restrict=name,server
    Restricts the returned json to the specified columns.
  • Parameter:   ignore - /character/get/730968?ignore=minions,mounts
    Ignores the specified columns from the returned json.

Achievements

/character/get/{id}/achievements

This returns the "Summary" page.

  • Example:   /character/get/730968/achievements

/character/get/{id}/achievements/{kind_id}

ID List: 1 = battle, 2 = character, 4 = items, 5 = synthesis, 6 = gathering, 8 = quests, 11 = explore, 12 = grand company, 13 = legacy.

Get achievements for a specific category.

  • Example:   /character/get/730968/achievements/2
