XIVSync API

What is this?

This API acts as a relay for obtaining data from the official game website: The Lodestone.

It provides a lightweight and clean response for various data from the API. The data is pulled straight from the API and never cached. This server is based in Tokyo, if your web server is in the US or EU, it is recommended to use the XIVPads API* instead.

* The XIVPads API does not parse the site, but provides pre-parsed data. Click here to read more.

What is required?

This API is REST-like, you just need to query the urls. They're all open and you should be good to go in any language. Responses are in JSON format. SDK are provided for Javascript and PHP.

Limitations?

There are no hard limits but please do no hit the server with a significant amount of queries. There is a soft build up of 2 queries per second before it prevents queries from the same source. This is enough for a single flow from a search page to a content page, eg: search character to select character.

If your server is in the US or EU then you may experience a 1-2 second delay per query, this is because of the hops your server has to make to connect to XIVSync and respond. Connecting to a Japan server from the US is slow! If this is not acceptable for your application, please use the XIVPads API.

Language support

The API can parse the lodestone in all 4 languages: English, French, German and Japanese. Just provide the query the language you would like with the tag: language={tag} to switch to that language. The default is English.
